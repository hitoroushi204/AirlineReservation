# ✈️ Airline Reservation System

Isang Desktop Application na binuo sa **Java** gamit ang **Swing** para sa pamamahala ng flight bookings at user administration.

## 🛠️ Tech Stack
- **Language:** Java
- **GUI:** Java Swing & AWT
- **Database:** MySQL
- **Library:** MySQL Connector J 9.6.0

## 📂 Project Structure
- `src/`: Laman ang lahat ng `.java` logic at `.form` GUI designs.
- `DBConnection.java`: Configuration para sa MySQL database connection.
- `Custom UI`: May kasamang Gradient Buttons at Transparent Panels.

## 🚀 Setup & Installation
1. **Database:** I-import ang iyong SQL file sa MySQL server.
2. **Library:** Siguraduhin na naka-add ang `mysql-connector-j-9.6.0.jar` sa iyong project libraries.
3. **Run:** Buksan sa IDE (NetBeans/VS Code) at i-run ang `AirlineReservation.java`.

## ✨ Features
- Login at Sign-up System
- Admin at User Dashboards
- Flight Booking at Ticket Management
- Modernong UI Design
